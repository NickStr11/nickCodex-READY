# Inbox

Минимальный входящий слой персональной ОС.

Файлы:
- `now.md` — один текущий фокус и ближайшие действия
- `backlog.md` — входящие идеи, хвосты и штуки без немедленного владельца

Правило простое:
- сейчас делаешь одно
- остальное либо ждёт в `backlog.md`, либо переезжает в нормальный долгоживущий файл
