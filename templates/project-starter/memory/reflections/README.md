# Reflections

Промежуточный слой для цикла `diary -> reflection -> rules`.

- Пиши сюда короткие reflection-заметки по повторяющимся паттернам.
- В `processed.log` отмечай diary/session файлы, которые уже разобраны.
- Устойчивые выводы переноси в узкий owner: `AGENTS.md`, `memory/PROJECT_CONTEXT.md`, `inbox/now.md`, `knowledge/` или project-local правила.

Не превращай эту папку в текущий статус проекта.
