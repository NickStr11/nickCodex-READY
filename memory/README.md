# Memory

Активный и временный слой проекта.

Что здесь лежит:
- `PROJECT_CONTEXT.md` — цель, рамка, активные направления
- `DEV_CONTEXT.md` — текущее состояние, свежие решения и рабочие ограничения
- `sessions/` — длинные handoff-ноты и сессионные хвосты

Что сюда не класть:
- долговечные дайджесты
- ресёрч, который должен переиспользоваться потом
- общие правила и личный профиль

Если задача не про текущий проект, эту папку можно не грузить.
