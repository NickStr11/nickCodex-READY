# Memory

Активный и временный слой проекта.

Что здесь лежит:
- `PROJECT_CONTEXT.md` — цель, рамка, активные направления
- `DEV_CONTEXT.md` — переходный hot-resume слой, пока новый diary-flow не устоится
- `diary/` — короткие записи завершённых сессий: что сделали, что решили, где остановились
- `sessions/` — длинные handoff-ноты и сессионные хвосты
- `reflections/` — короткие synthesis-заметки: какие повторяющиеся уроки перенесли в правила, skills или память

Что сюда не класть:
- долговечные дайджесты
- ресёрч, который должен переиспользоваться потом
- общие правила и личный профиль

Если задача не про текущий проект, эту папку можно не грузить.
