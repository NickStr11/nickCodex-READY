# AGENTS.md

Эта папка хранит живую память проекта, а не вечные правила.

## Как использовать

- Сначала смотри корневой `AGENTS.md`, потом файлы этой папки.
- `PROJECT_CONTEXT.md` — что это за проект, зачем он сейчас и куда идёт.
- `DEV_CONTEXT.md` — последний статус, свежие решения, known issues, ближайшие шаги.
- `sessions/` — длинные хвосты и handoff-заметки, которые не стоит держать в активном корне памяти.

## При правках

- Обновляй только когда реально поменялся статус, фокус или ограничения.
- Пиши коротко, с датами и без длинной истории ради истории.
- Не дублируй сюда правила из `rules/` и личный профиль из `aboutme.md`.
