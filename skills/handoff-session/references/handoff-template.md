# Handoff Template

## Update Checklist

Before finishing:
- refresh `memory/DEV_CONTEXT.md`
- decide whether this session needs a dated note in `memory/sessions/`
- rewrite `inbox/now.md` toward the next restart point
- move leftovers into `inbox/backlog.md`
