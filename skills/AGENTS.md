# AGENTS.md

Эта папка для reusable workflows, а не для разовых заметок.

## Что считать skill

- Один файл = один навык или один повторяемый workflow.
- Навык нужен только если его реально можно переиспользовать.
- Если задача разовая, не тащи её сюда.

## Как оформлять

- Сразу пиши, когда навык должен триггериться.
- Давай пошаговые инструкции и локальные ссылки вместо длинной теории.
- Явно указывай зависимости, права и внешние сервисы, если они нужны.
- Не хардкодь личные абсолютные пути без необходимости.

## При правках

- Сохраняй совместимость с форматом из `README.md`.
- Примеры добавляй только если они реально ускоряют работу агента.
- После структурных правок запускай `powershell -ExecutionPolicy Bypass -File scripts/validate-context-pack.ps1` из корня репозитория.
