# Vercel React Best Practices

Локальная упаковка skill'а с React/Next.js performance-правилами от Vercel.

Что здесь важно:
- `SKILL.md` — когда триггерить skill и как им пользоваться
- `AGENTS.md` — локальные инструкции для этой подпапки
- `rules/_sections.md` — карта разделов
- `rules/*.md` — отдельные performance-правила
- `metadata.json` — метаданные набора

Это не исходный source-repo Vercel, а установленный skill для чтения и применения правил.
