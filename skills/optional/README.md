# Optional Skills

Здесь лежат reusable, но не обязательные навыки.

Они нужны под конкретные классы задач:
- внешние исследования и документация
- видео и NotebookLM
- фронтенд-апгрейды
- отдельные writing или planning workflows

Если без навыка можно спокойно жить в новом project-local repo, это кандидат сюда, а не в `core/`.
