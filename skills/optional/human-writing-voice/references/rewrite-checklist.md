# Rewrite Checklist

Use this as a final pass, not as a rigid template.

## Keep

- core thought still intact
- concrete example still present
- natural roughness still present
- profanity preserved if it fits
- open ending preserved if the text wants one

## Cut

- polished AI transitions
- corporate tone
- fake certainty
- motivational fluff
- repeated structure across paragraphs

## Ask Before Final

- does this sound like a person thinking, not a system summarizing?
- did the rewrite become smarter-sounding than the source?
- did I accidentally clean up the only interesting part?
