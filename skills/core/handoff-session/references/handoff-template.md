# Handoff Template

## Update Checklist

Before finishing:
- write a short diary entry in `memory/diary/`
- decide whether this session needs a longer note in `memory/sessions/`
- rewrite `inbox/now.md` toward the next restart point
- move leftovers into `inbox/backlog.md`
- touch `memory/DEV_CONTEXT.md` only if a transitional hot-resume note is still needed
