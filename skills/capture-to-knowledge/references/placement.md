# Placement

## Routing Table

- Stable agent operating rule -> `AGENTS.md` or `rules/*.md`
- Current project state, next focus, latest decisions -> `memory/PROJECT_CONTEXT.md` or `memory/DEV_CONTEXT.md`
- Longer session tail -> `memory/sessions/`
- Reusable research or domain understanding -> `knowledge/`
- Personal facts and current life context -> `aboutme.md`
- Writing voice and stylistic constraints -> `writing-style.md`
- Deep motivations, anti-values, conflict patterns -> `deep-values.md`
- Philosophical lenses and interpretation frames -> `deep-philosophy.md`
- Repeatable workflow -> `skills/`
- Raw artifact or unresolved scrap -> `runtime/` or `inbox/backlog.md`

## Compression Rule

Do not preserve the whole conversation.
Preserve the smallest durable form that will still help later.
