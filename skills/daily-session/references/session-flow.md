## Backlog Triage

Sort each incoming item into one of four buckets:
- do now
- park for later
- capture into durable knowledge
- drop as noise

## `inbox/now.md` Shape

```md
# NOW

Обновлено: YYYY-MM-DD

## Current Focus
- One active track.

## Next Actions
- 1-3 concrete actions.

## Parked
- Everything that is not current.
```
