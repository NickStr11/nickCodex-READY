# AGENTS.md

Эта папка для рабочих артефактов, а не для инструкций.

## Что сюда класть

- сырые импорты
- выгрузки, html/json/xml/js дампы
- scratch-файлы и временные результаты
- research-output, который не должен жить в корне

## Как использовать

- Не читай `runtime/` целиком по умолчанию.
- Открывай только нужную подпапку под текущую задачу.
- Если артефакт стал долгоживущим знанием, перенеси его смысл в `knowledge/` или `memory/`, а не делай `runtime/` источником истины.
