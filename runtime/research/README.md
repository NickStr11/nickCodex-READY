# Research

Временный ресёрч-слой.

Сюда можно класть промежуточные заметки, ссылки и сводки, которые ещё не стали нормальным `knowledge/`.

Нормальная форма для папки:

```text
runtime/research/<topic-or-repo-slug>/
  summary.md
  links.md
  files.txt
```

Нормальная форма для recon-заметки:

```text
runtime/research/<topic-or-repo>-recon-YYYY-MM-DD.md
```

Шаблон: `RECON-TEMPLATE.md`.

Минимальные поля:
- source URL
- local clone/files
- checked commit/version
- source-backed claims
- transferable ideas
- reject list
- next actions

Если ресёрч стал долгоживущим знанием, смысл нужно вынести в `knowledge/` или `memory/`, а не держать `runtime/` источником истины.
