# Research

Временный ресёрч-слой.

Сюда можно класть промежуточные заметки, ссылки и сводки, которые ещё не стали нормальным `knowledge/`.

Нормальная форма:

```text
runtime/research/<topic-or-repo-slug>/
  summary.md
  links.md
  files.txt
```

Если ресёрч стал долгоживущим знанием, смысл нужно вынести в `knowledge/` или `memory/`, а не держать `runtime/` источником истины.
